==== 16.4.6.13 PSL-13 Product/Service Unit Cost (CP) 01967

Components: <Price (MO)> ^ <Price Type (ID)> ^ <From Value (NM)> ^ <To Value (NM)> ^ <Range Units (CWE)> ^ <Range Type (ID)>

==== 8.8.12.16 OM4-16 Specimen Preference (ID) 03311

Definition: This field indicates whether the Specimen/Attribute is Preferred or alternate for collection of the specimen. There can only be one occurrence of a Preferred or Alternate Specimen/Attribute for the code referenced in _OM4-6 Specimen_. For example, if two OM4 segments are received for specimen type of Serum, only one can be marked as Preferred.
